# bfban-discord-bot
[BFBAN Website](https://www.bfban.com)

----
BFBAN discord机器人支持bfban的大部分功能，包括但不限于查询玩家bfban状态、查询玩家职业信息、举报玩家、玩家申诉、网站回复消息等。

----

### 项目已完成初步编译,以下是待实现的功能

#### 1. 获取网站公告
#### 2. 获取被举报玩家列表
#### 3. 将查看bfban信息的单一搜索name更改为近似搜索被举报玩家名字及历史名字
#### 4. 发送消息

* 注意:以上待实现功能并未考虑实际操作可行性以及实现难度,不代表最终实现结果.
